# 🛒 Tamkeen E-Commerce & Admin Dashboard

A comprehensive, responsive, and fully functional E-commerce frontend application built with React. This project includes both a customer-facing storefront and a secure Admin Dashboard for managing products, categories, and statistics.

## ✨ Features

### 🛍️ Storefront (User Interface)
- **Dynamic Product Catalog:** Browse products, view details, and filter by categories.
- **Shopping Cart & Checkout:** Full cart functionality with order summary.
- **User Authentication:** Login, Register, and OTP verification flows.
- **Profile & Orders:** Users can manage their profiles and track order history.
- **Dark/Light Mode:** Seamless theme toggling for better user experience.
- **Bilingual Support (RTL/LTR):** Fully supports Arabic and English interfaces.

### ⚙️ Admin Dashboard
- **Overview Statistics:** Visual charts and KPI cards for revenue, orders, and users.
- **Product Management:** Full CRUD (Create, Read, Update, Delete) operations for products with image upload support.
- **Category Management:** Organize the store with dynamic category controls.
- **Server-Side Pagination:** Optimized data fetching for large catalogs.

## 🚀 Technologies Used
- **Framework:** React.js (Vite)
- **Routing:** React Router v6
- **Styling:** CSS Variables, React Bootstrap
- **Icons & Charts:** React Icons, Recharts
- **API Integration:** Fetch API with Token-based Authorization
